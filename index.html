SIFT (Scale Invariant Feature Transformation) algorithm was used to extract high pass responses from the photo and sketch of a corresponding face called as feature points using gradient based techniques. These feature points were described using a vector called as feature vector and matching pair of photo and sketch is the one which gives the nearest neighbour distance.
